1. life_animals.csv — 어떤 표인가

📦 오늘 쓸 데이터
  • 파일 = life_animals.csv · 여러 동물 종의 몸의 수치를 한 종에 한 줄씩 정리한 표
  • 열 = 이름 · 강(class) · 목(order) · 몸무게 · 최대수명 · 대사율 · 성숙일 · 임신일 · 한배새끼수 · 체온
  • 주소 = https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv · 다운로드 없이 주소로 바로 불러옴
🔎 데이터 출처 — AnAge
  • 원출처는 AnAge: The Animal Ageing and Longevity Database입니다. 노화 연구 자료 모음(Human Ageing Genomic Resources, HAGR)의 일부로, de Magalhães 연구진이 논문 근거로 관리하는 공개 데이터베이스예요.
  • 원본 사이트: genomics.senescence.info/species · 학술·교육 목적 이용.
  • 믿을 수 있는 이유 — 노화 생물학 연구자들이 종마다 수치의 논문 출처를 함께 기록해 관리하는 데이터베이스라, 값 하나하나가 어디서 왔는지 되짚을 수 있어요.
  • 수업에서는 척추동물 6개 강만 골라 열 이름을 알아보기 쉽게 정리한 사본을 씁니다. 원본을 그대로 쓰지 않았다는 점도 데이터를 다룰 때 밝혀야 할 정보예요.

큰 표는 통째로 보지 않습니다. 두 가지만 눈에 익히면 첫 인상을 잡을 수 있어요.

보는 법무엇을 보나
앞부분 몇 줄열 이름·값의 생김새 (표가 어떻게 생겼나)
요약통계개수·평균·최소·최대 (표의 첫 인상)

2. 표의 첫 인상 뽑기 (①②③)

① 샘플 프롬프트
동물 데이터(life_animals.csv)를 스트림릿으로 불러와서 표의 앞부분과 요약통계를 보여 줘.
종이 몇 개인지, 몸무게와 최대수명의 평균·최소·최대가 한눈에 보이게 해 줘.
주소는 여기야: https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv
  • 여러분 화면이 이런 모습이면 성공 · 글자까지 같을 필요 없음
② AI가 준 코드 예시
import streamlit as st
import pandas as pd

st.title("🧬 동물 데이터 첫 인상")

# 표 불러오기 (한 줄에 한 종)
url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv"
df = pd.read_csv(url)

# 앞부분 보기 — 표가 어떻게 생겼나
st.subheader("표의 앞부분")
st.dataframe(df.head())

# 요약통계 — 개수·평균·최소·최대 한눈에
st.subheader("숫자 요약")
st.dataframe(df.describe())

# 어떤 강(class)이 몇 종 있나
st.subheader("강(class)별 종 수")
st.dataframe(df["class"].value_counts())
  • 결과를 보고 → 세 박자대로 ③ 개선으로 한 걸음 더
③ 프롬프트 개선 예시
좋아! 여기에 하나 더 더해 줘.
열마다 값이 빠진 칸(관측이 없는 동물)이 몇 개인지도 함께 표로 보여 줘.
✔️ 요약통계 읽는 법
  • 개수(count) = 값이 몇 개 있나 (빈 칸은 안 셈)
  • 평균(mean) = 다 더해 개수로 나눈 값 · 최소·최대 = 가장 작은/큰 값
  • 몸무게 = 최소 0.5g부터 최대 1억 3,600만 g(대왕고래)까지 · 최대수명 = 최대 392년(그린란드상어)
🔎 오늘의 발견 — 표는 4,513종, 여섯 강
  • 전체 4,513종이 여섯 개 강(class)으로 나뉜다 → 새(Aves) 1,513 · 포유류(Mammalia) 1,349 · 경골어류(Teleostei) 806 · 파충류(Reptilia) 547 · 양서류(Amphibia) 181 · 연골어류(Chondrichthyes) 117
  • 그런데 열마다 값의 개수가 다르다 · 몸무게는 3,641종, 최대수명은 4,025종만 값이 있다
  • "어떤 칸은 왜 비어 있지?" → 이 물음이 전처리 차시로 이어진다

📚 오늘의 용어

오늘 나온 낱말 7개 — 외울 필요 없습니다 · 잊으면 여기로
용어뜻
행 / 열표의 가로 한 줄 / 세로 한 칸 (한 종이 행, 몸무게가 열)
강(class)생물 분류 단계 하나 — 포유류·새·파충류처럼 큰 갈래
정형 / 비정형행·열이 있는 표 / 이미지·글처럼 표가 아닌 데이터
결측값이 비어 있는 것 — 그 동물의 그 값은 측정되지 않았다는 뜻
요약통계개수·평균·최소·최대로 표의 첫 인상을 잡는 숫자
판다스(pandas)파이썬에서 표 데이터를 다루는 대표 도구
