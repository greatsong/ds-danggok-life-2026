0. 이 수업의 지도

한 학기 동안 우리는 동물 데이터 하나를 끝까지 다룹니다. 데이터를 바꿔 가며 얕게 훑는 대신, 하나를 깊이 파고들어 회귀·분류·군집 세 가지 방법을 이어서 배워요. 배우는 것은 코드 문법이 아니라 질문 세우기 · 데이터 감식 · 결과 해석 · 한계 인식입니다.

트랙차시무엇을 하나
회귀1~8동물의 몸무게로 "얼마나 오래 사나" 수명 숫자 예측
분류9~11동물이 "온혈일까 냉혈일까" 범주 맞히기
군집12~14몸의 프로파일이 비슷한 동물끼리 묶기
버퍼·선택15~16주기율표 확장 데모 · 팀 프로젝트 주제 정하기
💡 평가는 코드가 아니라 판단
  • 코드는 AI가 짠다 · 여러분은 주문하고, 읽고, 판단한다
  • 채점은 "왜 그 질문을 했나 · 데이터의 빈틈을 알아챘나 · 결과를 어디까지 믿나"

1. 데이터 과학이란

데이터 과학은 한마디로 데이터로 질문에 답하는 일입니다. 데이터에서 질문이 나오고, 질문이 답을 부르고, 그 답이 다시 새 질문으로 이어져요. 이 순환의 출발점은 언제나 좋은 질문 하나입니다.

  • 한 줄 정의 = 데이터로 질문에 답하는 일
  • 순환 = 데이터 → 질문 → 답 → 다시 질문
  • 이번 학기를 끌고 갈 질문 하나 → "몸집이 큰 동물은 정말 더 오래 사는가"
  • 이 한 문장이 8차시(만나기·시각화·전처리·모델·평가·배포·정리)를 전부 정한다

2. 바이브 코딩 리듬 — ①②③

이 수업에서 코드는 AI가 씁니다. 여러분은 자연어로 주문하고, 나온 결과가 맞는지 판단해요. 모든 실습은 세 박자로 흘러갑니다. 오늘은 리듬만 눈에 익히면 됩니다.

  • 모든 실습은 세 박자 → ① 샘플 프롬프트 → ② AI가 준 코드 → ③ 프롬프트 개선
  • ②는 선생님 화면으로 함께 보고, ③는 다음 차시부터 직접
① 샘플 프롬프트
동물 데이터(life_animals.csv)를 불러와서 표의 앞부분 다섯 줄만 보여 줘.
이 주소에 있어: https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv
  • 이렇게 말로 주문하면 AI가 코드로 답한다 · 여러분 화면이 이런 모습이면 성공(글자까지 같지 않아도 됨)
② AI가 준 코드 예시
import pandas as pd

url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv"
df = pd.read_csv(url)   # 표를 불러오기
df.head()               # 앞부분 다섯 줄 보기
  • 표가 뜨면 → "여기에 요약도 더해 줘" 하고 ③ 개선으로 이어 간다(다음 차시)
💡 프롬프트엔 코드 대신 정확한 낱말
  • 프롬프트에 read_csv 같은 코드 문법을 넣지 않는다 · 넣는 순간 이미 코드 아는 사람의 주문
  • 대신 파일 이름·주소·열 이름·도구 이름은 정확히 적는다 (예: life_animals.csv, 몸무게)

우리가 다룰 데이터는 modudata 저장소에 올라가 있어요. 주소는 https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv 입니다. 다운로드 없이 주소로 바로 불러옵니다.

3. 좋은 질문 세우기

좋은 분석은 좋은 질문에서 갈립니다. 데이터로 답할 수 있는 질문에는 세 가지 조건이 있어요.

조건뜻
측정 가능숫자나 값으로 잴 수 있다
데이터로 답 가능가진 데이터 안에서 답이 나온다
범위 명확무엇을·어떤 대상에서 보는지 딱 정해졌다
나쁜 질문고쳐 쓰면
어떤 동물이 가장 멋있나답 불가(취향) → 몸무게가 무거운 동물일수록 더 오래 사는가
동물은 얼마나 사나범위 없음 → 포유류는 몸무게로 최대 수명을 얼마나 설명할 수 있는가
✔️ 내 질문 카드 1장
  • 세 조건을 만족하는 내 질문을 한 문장으로 적는다
  • 몸무게-수명이 아니어도 좋음 · 대신 "데이터로 답할 수 있는가"만 스스로 점검
  • 이 카드가 학기 끝 팀 프로젝트의 첫 재료
🔎 오늘의 발견
  • 분석의 성패는 코드보다 질문에서 갈린다
  • "몸집이 큰 동물은 정말 더 오래 사는가" 한 문장이 앞으로 8차시를 끌고 간다
