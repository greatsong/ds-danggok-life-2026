정확도만 보면 놓치는 것

🎯 오늘 확인하는 것
  • 혼동행렬 해석표 — 83.5%라는 숫자를 세 각도에서 따져 본다

1. 확인 하나 — 아무것도 안 배워도 75%

  • 온혈·냉혈 종 수: 온혈 2,398 · 냉혈 790 (합 3,188)
  • 온혈이 전체의 2,398 ÷ 3,188 = 75.2%
  • 그러면 무조건 '온혈'로만 찍는 단순한 모델도 정확도 75.2%입니다
⚠️ 83.5%를 다시 보자
  • 기준선(전부 온혈)이 이미 75.2% · 내 모델 83.5%는 그 위 8.3%p일 뿐
  • 정확도는 가장 흔한 답만 찍어도 높게 나옵니다 — 한쪽이 많은 데이터에선 겉보기와 다릅니다

2. 확인 둘 — 정작 '냉혈'을 자주 놓친다 (①②③)

  • 정확도 하나로는 어느 갈래를 못 맞히는지 안 보입니다 → 혼동행렬과 재현율로 들여다봅니다
  • 재현율 = 실제 '냉혈'인 종 중에서 모델이 '냉혈'이라 맞힌 비율
① 샘플 프롬프트
지난 시간에 만든 온혈·냉혈 분류기를 이어서 분석해줘. 조건:

- 온혈과 냉혈을 서로 얼마나 헷갈리는지 혼동행렬을 표로 그려줘.
- 갈래마다 재현율을 계산해서, 특히 '냉혈'을 얼마나 잘 잡아내는지
  알려줘.
- 아무것도 안 배우고 무조건 '온혈'로만 찍었을 때의 정확도(기준선)도
  함께 계산해서 내 모델 정확도와 나란히 보여줘.
  • AI가 아래 모습의 코드를 줌 (똑같지 않아도 OK)
② AI가 준 코드 (예시)
import streamlit as st
import pandas as pd
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import cross_val_predict
from sklearn.metrics import confusion_matrix, classification_report, accuracy_score

url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv"
df = pd.read_csv(url).dropna(subset=["adult_weight_g", "max_longevity_yrs"])
df["is_warm"] = df["class"].isin(["Mammalia", "Aves"]).astype(int)

feats = ["adult_weight_g", "max_longevity_yrs",
         "female_maturity_days", "gestation_days", "litter_size"]
X = df[feats].fillna(df[feats].median())
y = df["is_warm"]

pred = cross_val_predict(RandomForestClassifier(random_state=42), X, y, cv=5)

# 혼동행렬 — 냉혈(0)·온혈(1)을 서로 얼마나 헷갈리나
names = ["냉혈", "온혈"]
cm = confusion_matrix(y, pred)
st.subheader("혼동행렬 (세로=실제, 가로=예측)")
st.dataframe(pd.DataFrame(cm, index=names, columns=names))

# 갈래별 성적표(정밀도·재현율)
st.subheader("갈래별 성적표")
st.text(classification_report(y, pred, target_names=names, zero_division=0))

# 기준선: 무조건 '온혈'로만 찍기
baseline = max(y.mean(), 1 - y.mean())
st.metric("내 모델 정확도", f"{accuracy_score(y, pred):.1%}")
st.metric("전부 '온혈'로 찍기", f"{baseline:.1%}")
⚠️ '냉혈' 재현율 = 0.66
  • 냉혈은 실제 세 종 중 한 종을 온혈로 잘못 봅니다(재현율 0.66)
  • 온혈은 잘 맞히니 전체 정확도는 높습니다 · 그런데 한쪽 갈래를 자주 놓치면 그 정확도만으로는 부족합니다
③ 프롬프트 개선 (예시)
좋아! 이어서 고쳐줘.
- '냉혈' 재현율을 크게 강조해서 보여줘.
- 냉혈인데 온혈로 잘못 맞힌 종을 몇 개만 예시로 뽑아 줘.

3. 확인 셋 — 정답을 미리 본 건 아닌지 (레이블 누수)

데이터에는 체온(body_temp_k) 열이 있습니다. 이걸 특징에 넣으면 정확도가 0.99 가까이 오릅니다(체온이 실제로 측정된 종만 남겨 재면 약 0.988). 하지만 이건 실력이 아니라 정답을 특징에 넣은 셈입니다.

⚠️ 왜 반칙인가 — 레이블 누수
  • 온혈·냉혈이란 곧 체온을 일정하게 유지하느냐의 문제입니다 · 체온을 알려 주는 건 정답을 미리 보여 주는 셈
  • 게다가 체온은 대부분의 종에서 측정돼 있지 않습니다(3,188종 중 422종만) · 실제로 쓸 수도 없는 특징입니다
  • 이렇게 정답에서 나온 정보로 올린 점수를 레이블 누수라 합니다

🔎 오늘의 발견 — 정확도만 보면 놓치는 세 가지

✔️ 83.5%만 보면 놓치는 것
  • 하나 — '온혈'만 찍어도 75.2%. 기준선을 안 보면 83.5%가 대단해 보인다
  • 둘 — '냉혈'은 세 종 중 한 종을 놓친다(재현율 0.66)
  • 셋 — 체온 열을 넣으면 0.99까지 오르지만 그건 정답을 미리 본 셈(레이블 누수)
