정리 — 상관은 관계, 법칙은 따로

"직선을 그을 수 있다고 원인을 아는 건 아니다. 그래도 데이터에서 법칙이 드러날 때가 있다."
회귀 8차시가 남긴 두 마음
🎯 오늘 하는 것
  • 상관·인과 메모 + 내 질문 확정 — 회귀 여정을 한 장으로 정리하고, 더 파 보고 싶은 질문을 정한다

0. 회귀 8차시 한눈에

차시한 일남긴 것
1~2질문 세우기 · 표 읽기좋은 질문이 분석을 정한다
3~4시각화 · 전처리없는 값에도 이유가 있다(측정 비용·로그로 펴기)
5~6모델링 · 평가와 한계언제 믿을 수 있나 — 세 조건(척도·집단·예외)
7배포내 예측기가 주소를 얻었다

1. 상관 ≠ 인과

  • 몸집이 큰 동물이 대체로 더 오래 삽니다. 둘은 함께 움직입니다(상관).
  • 그렇다고 몸집을 키우면 수명이 는다고 말하면 안 됩니다. 코끼리로 몸을 불려도 사람이 코끼리만큼 살진 않습니다.
  • 몸집과 수명은 대사 속도·심박·번식 전략 같은 숨은 요인들과 함께 얽혀 움직이는 셈입니다.
⚠️ 회귀는 관계를 볼 뿐, 원인을 증명하지 않는다
  • 회귀 직선은 "함께 움직인다"까지만 말합니다.
  • "몸집이 수명의 원인"이라는 결론은 회귀만으로 못 내립니다.
  • 왜 그런지는 회귀가 아니라 생물학 실험과 이론으로 따로 밝혀야 합니다.
💡 상관인데 인과로 착각한 문장 찾기
  • "바다에 사는 물고기가 오래 산다 → 동물을 물에 넣으면 오래 산다"
  • 어디가 틀렸나요? 깊은 바다의 낮은 수온·느린 대사라는 공통 요인이 빠졌습니다. 이런 문장을 스스로 만들어 봅시다.

2. 그래도 법칙이 드러날 때 — 클라이버 법칙 재발견

  • 상관이 인과는 아니지만, 데이터에서 깨끗한 규칙이 드러날 때가 있습니다.
  • 동물의 대사율(에너지를 쓰는 속도)과 체중을 로그로 놓고 직선을 그어 봅시다.
  • 생물학엔 클라이버 법칙이 있습니다 — 대사율은 체중의 0.75제곱에 비례한다는 오래된 법칙. 우리 데이터로 그 기울기가 나올까요?
① 샘플 프롬프트
동물의 대사율과 체중의 관계를 확인하는 코드를 만들어 줘.
- 같은 life_animals.csv에서 체중(body_mass_g)과 대사율(metabolic_rate_w)이
  둘 다 있는 종만 남겨 줘.
- 둘 다 로그(log10)로 바꾸고, 로그 체중으로 로그 대사율을 예측하는 직선을 학습해 줘.
- 그 직선의 기울기와 R²를 숫자로 보여 줘.
② AI가 준 코드 예시
import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

url = "https://raw.githubusercontent.com/greatsong/modudata/main/data/life_animals.csv"
df = pd.read_csv(url)

# 체중과 대사율이 둘 다 있는 종만 (대사율은 측정 비용이 커서 대부분 비어 있음)
k = df.dropna(subset=["body_mass_g", "metabolic_rate_w"]).copy()
k["log_mass"] = np.log10(k["body_mass_g"])      # 로그 체중
k["log_met"]  = np.log10(k["metabolic_rate_w"])  # 로그 대사율

model = LinearRegression()
model.fit(k[["log_mass"]], k["log_met"])

slope = model.coef_[0]
r2 = model.score(k[["log_mass"]], k["log_met"])
print(f"종 수: {len(k)}")           # 627
print(f"기울기: {slope:.3f}")        # 0.713
print(f"R^2: {r2:.2f}")             # 0.82
💡 생물에도 법칙이 있다
  • 우리 데이터로 나온 기울기는 0.713, 이론값 0.75에 놀랄 만큼 가깝습니다. R²는 0.82로 아주 잘 맞습니다.
  • 박테리아부터 코끼리까지, 몸이 클수록 단위 몸무게당 에너지는 덜 씁니다. 이 규칙이 생물 전체를 관통합니다.
  • 수업에서 우리가 손으로 그은 직선이 100년 된 생물학 법칙을 재발견했습니다. 데이터 과학이 과학과 만나는 지점입니다.
③ 프롬프트 개선 예시
좋아, 이어서 고쳐 줘.
- 로그 체중과 로그 대사율의 점, 그리고 학습한 직선을 plotly로 함께 그려 줘.
- 이론값 0.75 직선도 점선으로 겹쳐 그려서, 우리 기울기 0.713과 얼마나 가까운지 눈으로 보이게 해 줘.

3. 내 질문 확정하기

  • 1차시에 던진 "몸집이 큰 동물은 정말 더 오래 사는가"가 8차시를 끌고 왔습니다.
  • 이제 내가 더 파 보고 싶은 질문을 하나 정합니다 — 팀 프로젝트의 재료가 됩니다.
✔️ 좋은 질문의 조건 (1차시 복습)
  • 측정 가능 — 숫자나 범주로 답이 나오는가
  • 데이터로 답 가능 — 구할 수 있는 데이터가 있는가
  • 범위 명확 — 어느 집단·어느 척도인지 분명한가
📝 오늘의 정리 메모

  • 더 파 보고 싶은 내 질문 1개를 세 조건에 맞춰 적기.

🎒 오늘의 체크 & 링크

  • 상관(함께 움직임)과 인과(원인)는 다르다
  • 회귀는 관계를 볼 뿐 원인을 증명하지 않는다
  • 대사율과 체중을 로그로 그으면 기울기 0.713(이론 0.75)·R² 0.82 — 클라이버 법칙 재발견
  • 상관을 인과로 착각한 문장을 만들고 틀린 곳을 확인했다
  • 세 조건에 맞는 내 질문 1개를 확정했다
